How to Play

1. Enter Names

Type the full names of two people into the boxes above.

2. Cancel Common Letters

The game removes letters that appear in both names. For each common letter, one instance is removed from each name.

3. Count Remaining Letters

We count the total number of letters left over. This number is the key to your result!

4. Find the FLAMES outcome

Using the count, letters are eliminated from "FLAMES" one by one. The last letter remaining reveals your destiny!
